# 📊 Ledger Finance Tracker

A sleek, modern, and lightweight personal finance management tool built to track your daily income, expenses, and overall budget in real-time. 

🚀 **[View Live Demo](https://khaleelsyed8.github.io/NewLife/Ledger_Finance_Tracker.html)**

---

## ✨ Features

*   **Real-Time Balance Tracking:** Instantly calculates your current balance, total income, and total expenses.
*   **Dynamic Ledger:** Add, view, and organize financial transactions on the fly.
*   **Clean UI/UX:** A minimalist, distraction-free interface designed for seamless data entry and modern aesthetics.
*   **Fully Responsive:** Optimized for desktop, tablet, and mobile screens so you can track your budget on the go.
*   **Static & Fast:** Pure frontend implementation ensuring lightning-fast load times.

---

## 🛠️ Built With

*   **HTML5** – Structured semantic layout
*   **CSS3** – Modern responsive styling and layouts
*   **JavaScript (ES6+)** – Dynamic state management and transaction logic
